# PhotoFrame Pro 设计交付包

本包用于开发一个 FrameXT 替代工具：PhotoFrame Pro。

## 文件说明

- `design_spec.md`：UI 设计规范
- `requirements.md`：产品需求文档
- `assets/logo.svg`：主 Logo SVG
- `assets/logo_mark.svg`：图标版 Logo SVG
- `assets/ui_mockup_dark.png`：桌面端深色主题设计图

## 推荐技术栈

- Electron + React + TypeScript
- Sharp 图片处理
- exifr 读取 EXIF
- Zustand 状态管理
- Ant Design / Radix UI / Tailwind CSS
